# Tic Tac Toe Game
A simple and interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript**.  
This project demonstrates core frontend concepts like DOM manipulation, event handling, and game logic implementation.

## Features
- Two-player gameplay (X vs O)
- Turn-based logic
- Winner detection
- Tie detection
- Reset / New Game functionality
- Clean and simple UI

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla JS)
